This folder should contain the following files :

ConnectedComponent.cpp
ConnectedComponent.h

PGMimageProcessor.cpp
PGMimageProcessor.h

driver.cpp

as well as a makefile

this program can be run by typing make and then invoked by ./driver [options] <filename>

This Program also implements unit testing using the catch framework , to test this program type :'make test' into the terminal and it will run automatically.
the unit testing is implemented in ConnectedComponentTest.cpp
